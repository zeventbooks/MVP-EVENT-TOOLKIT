üîó Network Relationships

As a parent organization, you can manage relationships with child brands and configure which children are included in network analytics.

üí° How to Manage Network Relationships

To include or exclude a child organization from network reports:

  1. Open Google Apps Script Editor (Extensions ‚Üí Apps Script)
  2. Navigate to Config.gs
  3. Find the child tenant configuration (e.g., 'cbc' or 'cbl')
  4. Set includeInNetworkReports: true or false
  5. Save and deploy to see changes

üìñ Configuration Guide

How to Edit Configuration

  1. Open Google Apps Script Editor (Extensions ‚Üí Apps Script)
  2. Navigate to Config.gs file
  3. Edit the TENANTS array or ZEB object
  4. Save and deploy as a new version
  5. Refresh this page to see changes

Available Configuration Options

ZEB.APP_TITLE
Application name displayed across the platform
ZEB.BUILD_ID
Build version identifier for tracking deployments
TENANTS[].id
Unique tenant identifier used in URLs
TENANTS[].name
Display name for the tenant organization
TENANTS[].hostnames
Custom domain mapping for tenant-specific URLs
TENANTS[].scopesAllowed
Allowed scopes: events, leagues, tournaments, sponsors
