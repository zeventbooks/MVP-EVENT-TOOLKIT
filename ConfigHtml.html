üîó Brand Portfolio

As a parent organization, you can manage relationships with child brands and configure which children are included in portfolio analytics.

üí° How to Manage Brand Portfolio

To include or exclude a child organization from portfolio reports:

  1. Open Google Apps Script Editor (Extensions ‚Üí Apps Script)
  2. Navigate to Config.gs
  3. Find the child brand configuration (e.g., 'cbc' or 'cbl')
  4. Set includeInPortfolioReports: true or false
  5. Save and deploy to see changes

üìñ Configuration Guide

How to Edit Configuration

  1. Open Google Apps Script Editor (Extensions ‚Üí Apps Script)
  2. Navigate to Config.gs file
  3. Edit the TENANTS array or ZEB object
  4. Save and deploy as a new version
  5. Refresh this page to see changes

Available Configuration Options

ZEB.APP_TITLE
Application name displayed across the platform
ZEB.BUILD_ID
Build version identifier for tracking deployments
BRANDS[].id
Unique brand identifier used in URLs
BRANDS[].name
Display name for the brand organization
BRANDS[].hostnames
Custom domain mapping for brand-specific URLs
BRANDS[].scopesAllowed
Allowed scopes: events, leagues, tournaments, sponsors
